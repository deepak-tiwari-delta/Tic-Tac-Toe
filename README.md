# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

Add your live demo link here after deployment.

## 📌 Features

- Two-player gameplay
- Winner detection
- Game reset functionality
- Sound effects for moves and game events
- Responsive and user-friendly interface
- Clean and simple UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/deepak-tiwari-delta/Tic-Tac-Toe.git
```

2. Open the project folder

```bash
cd Tic-Tac-Toe
```

3. Open `index.html` in your browser.
